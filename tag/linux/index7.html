    $ git config --global user.name "Your Name"

    $ mkdir git
$ cd git
$ mkdir "目录名"
$ cd "目录名"

    1.增加sudo用户组

    $ su

    转到root用户，输入密码 

    # visudo

    在root ALL=(ALL) ALL下增加一行 

    yourname ALL=(ALL) ALL

    如果不想每次输入密码更改为 

    yourname ALL=(ALL) NOPASSWD:ALL

:wq #保存退出  
exit #回到一般用户

    2.备份的主题，图标,字体拷贝到home目录
