  4. python复习--装饰器

    四 28 七月 2016
    By honmaple

    In Python.

    tags: python装饰器

    一个装饰器

    from functools import wrapper

def log(func):
    @wraps(func)
    def wrapper(*args, **kw):
        print('call %s():' % func.__name__)
        return func(*args, **kw)
    return wrapper

    或者针对带参数的decorator：

    def log(text):
    def decorator(func):
        @wraps(func)
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator
