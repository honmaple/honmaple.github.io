archlinux安装steam

一 25 四月 2016 | By honmaple | In Linux | 0f linux archlinux steam

最近想要玩会儿游戏，但是又不想切换到windows,所以准备安装steam,在linux下玩游戏, 参考文档Steam

准备工作

因为我的是64位操作系统,所以需要启用 multilib

$ sudo vim /etc/pacman.conf
# 取消下面注释
[multilib]
Include = /etc/pacman.d/mirrorlist

双显卡bumblebee

我的又是双显卡,需要安装32位驱动

lib32-nvidia-utils
lib32-mesa-libgl
lib32-mesa

注意 除此之外，参考 bumblebee报错信息 还需要安装 virtualgl-libs:i386 ( arch上是lib32-virtualgl ) 否则报错

ERROR: ld.so: object 'libdlfaker.so' from LD_PRELOAD cannot be preloaded: ignored.

安装steam

$ sudo pacman -Syu steam
