GNS3笔记

2021年05月11日 周二 linux gns3

快速导航

  1. GNS3 VM导入
  2. 禁用广告与数据采集
  3. 启用basic auth

GNS3 VM导入

1virt-install  --name gns3  --memory 8192  --vcpus 4  --disk /home/kvm/GNS3/GNS3VM-disk001.qcow2  --disk /home/kvm/GNS3/GNS3VM-disk002.qcow2  --import --virt-type kvm --os-type linux --os-variant ubuntu18.04 --network default --graphics vnc,listen=0.0.0.0,password=foobar,port=5910 --noautoconsole --hvm

禁用广告与数据采集

  • 禁用错误报告

    找到 ~/.config/GNS3/2.2/gns3_server.conf, 在 [Server] 块中添加 

    1[Server]
2...
3report_errors = False

  • 禁用google分析

    找到 /usr/local/lib/python3.8/dist-packages/gns3server/static/web-ui/index.html , 注释google分析相关代码 

     1<!-- <script async="" src="https://www.googletagmanager.com/gtag/js?id=G-5D6FZL9923"></script> -->
 2<script>
 3  window.dataLayer = window.dataLayer || [];
 4  function gtag() {
 5  dataLayer.push(arguments);
 6  }
 7  gtag('js', new Date());
 8
 9  // gtag('config', 'G-5D6FZL9923');
10</script>

  • 禁用错误跟踪

    找到 /usr/local/lib/python3.8/dist-packages/gns3server/static/web-ui 目录下的 main.xxxx.js 

    1sudo sed -i 's%https://d8be3a98530f49eb90968ff396db326c@o19455.ingest.sentry.io/842726%%g' main.2f0314a517dded67879c.js

  • 禁用广告

    同样找到 /usr/local/lib/python3.8/dist-packages/gns3server/static/web-ui 目录下的 main.xxxx.js 

    1sudo sed -i 's%https://servedbyadbutler.com/adserve/;ID=165803;size=0x0;setID=371476;type=json;%%g' main.2f0314a517dded67879c.js

然后重启服务即可 sudo systemctl restart gns3

启用basic auth

找到 ~/.config/GNS3/2.2/gns3_server.conf, 在 [Server] 块中添加 

1[Server]
2...
3auth = True
4user = gns3
5password = gns3
