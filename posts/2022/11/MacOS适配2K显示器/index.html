MacOS适配2K显示器

2022年11月24日 周四 阅读约 2 分钟 life macos

快速导航

  1. 安装
  2. 配置

最近购买了一个2K显示器，到手后才发现macos默认对2K分辨率支持不足，导致字体很小，无法像windows对字体进行缩放，尤其是终端字体，简直不忍直视。网上一搜才知道，这已经是陈年问题，网上也有很多解决办法，本次采用 BetterDisplay 开启 HiDPI， 不用关闭 SIP

安装

安装可以直接从 BetterDisplay 下载，手动安装，不过我推荐使用 brew 安装，这样对于软件的管理也更方便 

 1──╼ brew info BetterDisplay
 2==> betterdisplay: 1.3.4 (auto_updates)
 3https://betterdisplay.pro/
 4/usr/local/Caskroom/betterdisplay/1.3.4 (127B)
 5From: https://mirrors.ustc.edu.cn/homebrew-cask.git
 6==> Name
 7BetterDisplay
 8==> Description
 9Display management tool
10==> Artifacts
11BetterDisplay.app (App)
12==> Analytics
13install: 553 (30 days), 1,329 (90 days), 2,166 (365 days)
14──╼ brew install betterdisplay

安装后打开可能会提示更新，忽略即可

配置

在菜单栏选择 betterdisplay -> 点击齿轮(设置) -> Displays, 然后选择显示器，比如我这选择 AG273QG3R3B, Color LCD 则是笔记本自带屏幕

勾选 Edit the default system configuration of this display 和 Set up native smooth resolution scaling, 然后点击 Apply Changes 按钮, 输入管理员密码并且重启, 重启完成后选择 系统设置 -> 显示器, 选择适合自己的分辨率(这里发现只能选择4种，更多分辨率可通过betterdisplay设置)
