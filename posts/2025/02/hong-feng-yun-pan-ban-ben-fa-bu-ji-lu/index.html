v1.2.0 (2025-11-06)

新增

  • 服务增加验证机制及优化服务启动
  • 可开启webdav服务

修复

  • CFB文件解密

优化

  • 缩略图存储路径
  • 界面显示问题若干

v1.1.0 (2025-10-24)

新增

  • 访问频率限制
  • 文件加密支持 OFB 和 CTR

修复

  • 大文件上传卡顿
  • 大文件流式加密和解密
  • 开启缓存时文件列表刷新策略

优化

  • 通知从 snackbar 切换至 smartDialog

v1.0.9 (2025-06-12)

新增

  • 115网盘(不支持上传)
  • 夸克网盘(不支持上传)
  • S3增加 force_path_style 和 session_token 参数

修复

  • 大文件上传错误
  • Github文件获取错误
  • 阿里云OSS需设置 force_path_style 为 false

v1.0.8 (2025-05-01)

新增

  • ios支持
  • 文件排序支持目录优先
  • 移动端视频播放支持mkv格式
  • 支持 Github 和 Github Release 存储(只读)
  • 增加启动Logo页面

修复

  • 应用置于后台时并重新打开时服务断开

优化

  • 视频、音频、图片的预览
  • MacOS端bookmark优化
  • 移除部分不必要的包依赖

v1.0.5 (2025-02-16)

新增

  • 桌面端增加右键菜单
  • 应用数据可选备份至云盘或者本地

修复

  • 重试任务时应用退出
  • 任务列表图标颜色同主题保持一致
  • MacOS沙盒模式下本地存储访问受限

优化

  • MacOS下应用图标使用白色背景

v1.0.4 (2025-02-03)

新增

  • 文件拖拽上传支持文件夹
  • 文件列表信息缓存
  • S3存储增加 list_version 选项
  • SMB存储增加 domain 选项

依赖

  • Flutter版本升级至3.27.3

修复

  • Alist根目录无法上传/新建目录/移动/复制
  • 拖拽文件导致重复上传 issue25

优化

  • 添加/修改存储时，更多设置使用卡片分隔不同的选项

v1.0.3 (2025-01-24)

新增

  • 新增帮助文档，并使用外部浏览器查看

修复

  • 修复自定义挂载目录在非根目录/时的问题

优化

  • 使用 flex_color_scheme 优化界面主题和颜色

v1.0.2 (2025-01-09)

新增

  • windows支持

移除

  • 同步备份功能(bug较多无法使用)

优化

  • 桌面端UI

  • 文件加密和压缩
  • 桌面端文件拖拽上传(仅文件)

优化
