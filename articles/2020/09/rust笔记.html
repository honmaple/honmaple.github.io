安装

  • 安装 

    └──╼ brew install rustup
└──╼ rustup-init
......
1) Proceed with installation (default)
2) Customize installation
3) Cancel installation
>1

  • 配置 rust 目录 

    └──╼ mkdir $HOME/repo/rust
└──╼ mv $HOME/.rustup $HOME/repo/rust/rustup
└──╼ mv $HOME/.cargo $HOME/repo/rust/cargo

    写入 .bashrc 文件 

    export RUSTUP_HOME=$HOME/repo/rust/rustup
export CARGO_HOME=$HOME/repo/rust/cargo
export PATH=$PATH:$HOME/repo/rust/cargo/bin

  • 验证 

    └──╼ source .bashrc
└──╼ cargo --version
cargo 1.46.0 (149022b1d 2020-07-17)
