postgresql学习1——安装

2015年12月22日 周二 | 分类 Tech | 标签 sql postgresql linux

快速导航

  • archlinux
  • centos

最近从sqlite转到postgresql,虽然postgresql比sqlite更加复杂

archlinux

参考资料

 1$ sudo pacman -S postgresql
 2$ sudo mkdir /var/lib/postgres/data #如果已存在则可以不建立
 3$ sudo chmod -R postgres:postgres /var/lib/postgres/data
 4由于我安装系统时没有生成en_US_UTF-8本地化文件(如果已经生成了的就
 5不必再生成)
 6$ sudo vim /etc/locale.gen #找到en_US_UTF-8，注释掉
 7$ locale-gen
 8
 9然后运行
10$ sudo -i -u postgres #或者使用root账户su - postgres
11$ initdb --locale en_US.UTF-8 -E UTF8 -D '/var/lib/postgres/data'
12$ createuser -i #输入你的账户名称,并给管理员权限
13
14最好给postgres用户设置密码
15(注意不是linux系统帐号)
16# su - postgres
17$ psql
18>ALTER USER postgres WITH PASSWORD 'yourpasswd';

centos

参考资料

到这里 下载所需要的rpm安装包
或者直接

 1# yum localinstall http://yum.postgresql.org/9.4/redhat/rhel-6-x86_64/pgdg-centos94-9.4-1.noarch.rpm
 2# yum list postgres*
 3# yum install postgresql94-server
 4# yum install postgresql94-contrib
 5
 6# service postgresql initdb  #初始化数据库
 7# service postgresql start #启动数据库
 8# chkconfig postgresql on  #将数据库服务加入启动列表
 9修改PostgreSQL 数据库用户postgres的密码
10(注意不是linux系统帐号)
11# su - postgres
12$ psql
13>ALTER USER postgres WITH PASSWORD 'yourpasswd';
