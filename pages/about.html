博客历史

  • 2015-09

    创建, 使用自己写的 maple-blog (flask+nginx)作为博客系统

  • 2015-10

    厌倦了总是需要登陆才能发布，遂迁移到 GitHub Page ，选用基于 Python 的静态博客系统 Pelican, 曾使用域名 honmaple.gq, honmaple.tk

  • 2016-03

    域名更改为 honmaple.org

  • 2016-07

    博客主题由 subtle 更改为自己定制的主题，取名为maple-theme

  • 2017-10

    域名更改为 honmaple.me

  • 2021-04

    博客更名为 紅楓吟

  • 2023-02

    博客系统切换至 snow，这是我使用 Go 开发的一个轻量级静态博客生成器，并保留原来的主题

  • 2023-04

    评论系统从 utterances 切换至 giscus，主要是因为 utterances 加载速度比较慢，而且是基于 issue，而 giscus 基于 Discusstion，可以进行单独回复(楼中楼功能)

  • 2023-06

    主题移除了 Bootstrap 的支持，原本想要使用原子类 Tailwind CSS, 后面发现需要自定义的地方太多，故采用 SASS 完全自定义，下一步将继续移除 jQuery
