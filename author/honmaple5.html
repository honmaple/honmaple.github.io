    def wrapper(*args, **kw):
        print('call %s():' % func.__name__)
        return func(*args, **kw)
    return wrapper

    tags: tarlinux

    使用--exclude + 需要排除的目录

    示例: 

    tar zcvf hello.tar.gz --exclude="**/__pycache__" maple/ manager.py

    注意 : --exclude 参数需要在要打包的文件或目录前,否则不起作用,另外排除的目录后不能加/
